Project on Console

Visual Studio Code is an application which helps us to do some advanced coding. First we have to download this app, then create a folder and open it with VSC. Then you can start writing your code into it.

Console screen is a screen in which we can view our errors and run JavaScript. It is open by right clicking on the screen and clicking on inspect. Then on the upper navigator, there will be an option of console or you can open it by the shortcut Ctrl+Shift+I. Printing on console is simple. In your code, type console.log() and then type whatever you want to be printed in the console inside your brackets.
